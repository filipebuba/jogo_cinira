# Cinira: Jornada Universitaria

Prototipo de jogo educativo desenvolvido para trabalho de faculdade.

## Ideia do projeto

O jogo coloca o estudante em uma jornada por regioes de uma universidade. O jogador faz login, escolhe um personagem e explora o campus por meio de desafios. Cada regiao possui 4 niveis; ao concluir todos os niveis de uma regiao, uma nova area e desbloqueada.

## Telas implementadas

- Login do jogador
- Definicoes e informacoes do projeto
- Selecao de personagens
- Mapa de regioes
- Tela de desafio
- Tela de conclusao da fase

## Regioes do jogo

1. Entrada do Campus
2. Corredores
3. Biblioteca
4. Portao da Aprovacao

## Objetivo pedagogico

Usar gamificacao para apresentar conceitos de rotina universitaria, organizacao academica, uso de espacos da faculdade, pesquisa, trabalho em equipe e progresso por etapas.

## Como abrir

Abra o arquivo `index.html` no navegador ou use um servidor local:

```bash
python -m http.server 4173 --bind 127.0.0.1
```

Depois acesse:

```text
http://127.0.0.1:4173/index.html
```
